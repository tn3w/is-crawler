Tiny, zero-dependency crawler detection for Python. One function call, regex-powered, blazing fast.

User-Agent Detector

How it works

Four layered regex checks run in sequence. If any signal fires, the user-agent is classified as a crawler.

1

Bot signals

Matches keywords like bot, crawl, spider, scrape, URLs, and email patterns common in crawler identifiers.

2

Missing browser signature

Real browsers include engine tokens like Mozilla/, WebKit, or Gecko. Absence is a strong crawler signal.

3

Bare compatible

Detects (compatible; ...) strings without real OS/browser tokens - a pattern used by bots mimicking browsers.

4

Known tools

Identifies specific tools: Lighthouse, Playwright, Selenium, wget, and patterns like domain-prefixed agents.

example.py
from is_crawler import crawler_name, is_crawler

ua = request.headers.get("User-Agent", "")

if is_crawler(ua):
    print("Blocked crawler", crawler_name(ua))  # e.g. "Googlebot"
else:
    print("Human visitor")   # False for real browsers
