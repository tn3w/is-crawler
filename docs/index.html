III. Heuristics

Five checks.
First hit wins.

Short-circuit
a
01 / 05
Bot signals
bot, crawl, spider, scrape, headless. URL and email patterns common in crawler identifiers.
b
02 / 05
Missing browser
Real browsers always carry WebKit, Gecko, or Trident. Absence is a strong signal.
c
03 / 05
Bare compatible
(compatible; …) without real OS or browser tokens. A classic bot pattern mimicking browsers.
d
04 / 05
Known tooling
playwright, selenium, wget, lighthouse, sqlmap, and other named stacks.
e
05 / 05
URL in UA
An embedded http:// or https:// inside the user-agent. A near-universal bot convention.
IV. Database

Known crawlers,
lazily loaded.

On first call
646
curated entries

Built on monperrus/crawler-user-agents with supplemental tags. Each entry carries a URL, a description, and one or more tags. The database parses only on the first call to crawler_info.

Tag vocabulary
search-engine ai-crawler seo social-preview advertising archiver feed-reader monitoring scanner academic http-library browser-automation
V. Surface

Seven functions.
That is all.

API reference
Function Returns Purpose
is_crawler(ua) bool Heuristic detection. Fast, no DB.
crawler_signals(ua) list[str] Which of the five rules matched.
crawler_info(ua) CrawlerInfo | None DB lookup: url, description, tags.
crawler_has_tag(ua, tag) bool Tag membership. Accepts str or list.
crawler_name(ua) str | None Product name extracted from UA.
crawler_version(ua) str | None Version extracted from UA.
crawler_url(ua) str | None URL embedded in UA, if any.
VI. In practice

Usage & speed.

Example · benchmark
example.py python 3.9+
from is_crawler import is_crawler, crawler_info, crawler_has_tag

ua = "Googlebot/2.1 (+http://www.google.com/bot.html)"

is_crawler(ua) # True
crawler_has_tag(ua, "search-engine") # True
crawler_has_tag(ua, ["ai-crawler", "seo"]) # False

info = crawler_info(ua)
info.url # '…/bot.html'
info.description # "Google's main crawler"
info.tags # ('search-engine',)
