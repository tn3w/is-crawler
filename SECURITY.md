# Security Policy

## Supported Versions

Only the latest release on PyPI receives security fixes.

## Reporting a Vulnerability

**Do not open a public issue.**

Report vulnerabilities via [GitHub private security advisory](https://github.com/tn3w/is-crawler/security).

Include:

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

You will receive a response within 72 hours. If confirmed, a patch will be released promptly and you will be credited unless you prefer otherwise.
